# Milestone v13.0: Standard Library Extension

**Status:** IN PROGRESS
**Phases:** 93-95

## Phases

### Phase 93: String Module Extension
- `String.split`, `String.indexOf`, `String.replace`, `String.toUpper`, `String.toLower`
- `String.substring` (alias for string_sub), `String.join` (alias for concat)
- Built-in functions in Eval.fs + Prelude/String.fun wrappers

### Phase 94: List Module Extension
- `List.init`, `List.find`, `List.findIndex`, `List.partition`, `List.groupBy`
- `List.scan`, `List.replicate`, `List.collect`, `List.pairwise`
- Pure Prelude implementations (no Eval.fs changes needed)

### Phase 95: Array Module Extension + Misc
- `Array.zip`, `Array.find`, `Array.findIndex`, `Array.contains`
- `List.zip` already exists; mirror for Array
- flt integration tests for all new functions
