# Requirements: LangThree v8.0

**Defined:** 2026-03-30
**Core Value:** FunLexYacc 호환성을 위해 module-level 함수 선언에 F# 스타일 타입 어노테이션 지원

## v8.0 Requirements

### Parameter Type Annotations

- [x] **PARAM-01**: `let` 선언에서 파라미터 타입 어노테이션 허용 — `let f (x : int) y (z : bool) = ...` (plain과 annotated 혼합 가능)
- [x] **PARAM-02**: `let rec ... and ...` 상호 재귀 선언에서 파라미터 타입 어노테이션 허용

### Return Type Annotations

- [x] **RET-01**: `let` 선언에서 반환 타입 어노테이션 허용 — `let f x : int = ...`
- [x] **RET-02**: 파라미터 타입 + 반환 타입 동시 사용 — `let f (x : int) : bool = ...`

### Angle Bracket Generics

- [x] **GEN-01**: `type` 선언에서 앵글 브래킷 사용 — `type Result<'a> = Ok of 'a | Error of string`
- [x] **GEN-02**: 타입 표현식에서 앵글 브래킷 사용 — 람다/어노테이션에서 `Result<'a>` 타입 참조
- [x] **GEN-03**: 기존 후위(postfix) 구문 호환 유지 — `'a option`, `int list` 등 기존 구문 정상 동작
