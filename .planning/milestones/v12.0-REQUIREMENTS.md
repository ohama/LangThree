# Requirements: v12.0 Type Class Extensions

**Defined:** 2026-04-01
**Core Value:** 타입 클래스가 실용적으로 동작 — 컨테이너 타입에 대한 인스턴스, 클래스 계층, 자동 도출

## v12.0 Requirements

### Constrained Instances
- [ ] **CI-01**: `instance Show 'a => Show (list 'a)` 선언 가능
- [ ] **CI-02**: `show [1; 2; 3]`이 `"[1; 2; 3]"` 반환 (재귀적 제약 해결)
- [ ] **CI-03**: `instance Eq 'a => Eq (list 'a)` 선언 가능
- [ ] **CI-04**: 인스턴스 해결이 유니피케이션 기반 (직접 비교 → unify)
- [ ] **CI-05**: 재귀적 제약이 해결 불가능하면 적절한 에러 메시지

### Superclass Constraints
- [ ] **SC-01**: `typeclass Eq 'a => Ord 'a` 선언 가능 (슈퍼클래스 구문)
- [ ] **SC-02**: Ord 인스턴스는 자동으로 Eq 인스턴스 요구
- [ ] **SC-03**: Ord 제약에서 Eq 메서드 접근 가능

### Automatic Deriving
- [ ] **DRV-01**: `type Color = | Red | Green | Blue deriving Show` 구문 지원
- [ ] **DRV-02**: Show 도출 — 생성자 이름을 문자열로 변환하는 show 함수 자동 생성
- [ ] **DRV-03**: Eq 도출 — 구조적 동등성 비교하는 eq 함수 자동 생성
- [ ] **DRV-04**: 인자를 가진 생성자의 도출은 제약 조건부 인스턴스 필요 (CI-01 전제)

## Out of Scope
| Feature | Reason |
|---------|--------|
| Multi-parameter type classes | 복잡도 대비 실용성 낮음 |
| Functional dependencies | Multi-param 전제 |
| Default method implementations | v12.1에서 고려 |
| Higher-kinded types (Functor, Monad) | 별도 대형 마일스톤 |
