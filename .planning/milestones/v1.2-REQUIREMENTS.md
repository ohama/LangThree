## Milestone v1.2: Practical Language Features

### Pipe & Composition (PIPE)

- [x] **PIPE-01**: `|>` 파이프 연산자 — `expr |> func` 를 `func expr`로 평가
- [x] **PIPE-02**: `>>` 합성 연산자 — `f >> g` 를 `fun x -> g (f x)`로 평가
- [x] **PIPE-03**: `<<` 역합성 연산자 — `f << g` 를 `fun x -> f (g x)`로 평가

### Unit Type (UNIT)

- [x] **UNIT-01**: `()` 리터럴 파싱 및 평가 — unit 값 표현
- [x] **UNIT-02**: `unit` 타입 — 타입 추론에서 unit 타입 지원
- [x] **UNIT-03**: 부수효과 시퀀싱 — `let _ = expr1 in expr2` 또는 세미콜론 시퀀싱

### String Operations (STR)

- [x] **STR-01**: `string_length` — 문자열 길이 반환
- [x] **STR-02**: `string_concat` — 두 문자열 결합 (+ 연산자 외 함수 형태)
- [x] **STR-03**: `string_sub` — 부분 문자열 추출
- [x] **STR-04**: `string_contains` — 문자열 포함 여부
- [x] **STR-05**: `to_string` — int/bool을 문자열로 변환
- [x] **STR-06**: `string_to_int` — 문자열을 int로 변환

### Printf (PRINT)

- [x] **PRINT-01**: `print` — 값을 stdout에 출력 (줄바꿈 없이)
- [x] **PRINT-02**: `println` — 값을 stdout에 출력 (줄바꿈 포함)
- [x] **PRINT-03**: `printf` — 포맷 문자열로 출력 (`%d`, `%s`, `%b`)
