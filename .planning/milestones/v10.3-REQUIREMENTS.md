# Requirements: v10.3 Blank Line Tolerance in Indented Blocks

**Defined:** 2026-04-01
**Core Value:** 들여쓰기 블록 내부에서 빈 줄을 허용하여 코드 가독성 보장

## v10.3 Requirements

### IndentFilter

- [ ] **BL-01**: module 본문 내 선언 사이 빈 줄 허용 (`let`—`let`, `type`—`let`, `type`—`instance`, `type`—`type`)
- [ ] **BL-02**: 중첩 모듈 내부 빈 줄 허용
- [ ] **BL-03**: let 본문 내 `let`—`let` 사이 빈 줄 허용
- [ ] **BL-04**: match 키워드와 첫 `|` arm 사이 빈 줄 허용
- [ ] **BL-05**: 기존 동작(최상위, namespace) 유지 — 회귀 없음

## Out of Scope

| Feature | Reason |
|---------|--------|
| parse error 위치 정보 | 별도 이슈, fsyacc 파서 레벨 수정 필요 |
| 연속 빈 줄 2개 이상 | 1개 빈 줄만 허용, 2개 이상은 현행 유지 |
