## Milestone Summary

**Key Decisions:**
- BracketDepth in IndentFilter for bracket-aware indentation suppression
- LetPatDecl for module-level pattern binding (not grammar-level desugaring)
- Char literal rules placed BEFORE type_var in Lexer.fsl (longest-match disambiguation)
- Comparison widening uses synth-then-unify-then-match (not inferBinaryOp)
- Expression LetRec uses BuiltinValue + mutable envRef (trampoline fix)
- Mutable delegate pattern for TypeCheck.fs → Prelude.fs compile-order bridging
- MOD-05 recEnv scoping already correct — no fix needed

**Issues Resolved:**
- Multi-line lists broke due to spurious INDENT/DEDENT inside brackets
- Local let rec failed inside lambda bodies (trampoline lost self-binding)
- else+keyword parsed incorrectly (INDENT emitted before ELSE)

**Technical Debt Incurred:**
- span.FileName always empty in fsyacc-generated positions — workaround via mutable currentFile
- File import uses mutable HashSet for cycle detection (functional Set would require threading)
