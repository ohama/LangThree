**Status:** ✅ SHIPPED 2026-03-31

## Overview

v9.1 fixes the AST Span zeroing bug where all position information was lost through the IndentFilter pipeline. The fix embeds position data in each token (PositionedToken), preserves it through filtering, and updates lexbuf positions in the custom tokenizer.
