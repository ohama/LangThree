**Bidir.fs (type inference):** Add synth cases for the three new operators in the `synth` function. Place them after the existing binary operator cases (after `Cons`).

For `PipeRight(left, right, span)` — type as reversed application:
```fsharp
| PipeRight (left, right, span) ->
    // x |> f  ===  f x
    // Infer x type, infer f type, unify f with (x_type -> result)
    let s1, leftTy = synth ctorEnv recEnv ctx env left
    let s2, rightTy = synth ctorEnv recEnv ctx (applyEnv s1 env) right
    let resultTy = freshVar()
    let s3 = unifyWithContext ctx [] span (apply s2 rightTy) (TArrow(apply s2 leftTy, resultTy))
    (compose s3 (compose s2 s1), apply s3 resultTy)
```

For `ComposeRight(left, right, span)` — `f >> g = fun x -> g(f(x))`:
```fsharp
| ComposeRight (left, right, span) ->
    // f >> g : ('a -> 'b) -> ('b -> 'c) -> ('a -> 'c)
    let s1, leftTy = synth ctorEnv recEnv ctx env left
    let s2, rightTy = synth ctorEnv recEnv ctx (applyEnv s1 env) right
    let a = freshVar()
    let b = freshVar()
    let c = freshVar()
    let s3 = unifyWithContext ctx [] span (apply s2 leftTy) (TArrow(a, b))
    let s4 = unifyWithContext ctx [] span (apply s3 rightTy) (TArrow(apply s3 b, c))
    let finalS = compose s4 (compose s3 (compose s2 s1))
    (finalS, TArrow(apply finalS a, apply s4 c))
```

For `ComposeLeft(left, right, span)` — `f << g = fun x -> f(g(x))`:
```fsharp
| ComposeLeft (left, right, span) ->
    // f << g : ('b -> 'c) -> ('a -> 'b) -> ('a -> 'c)
    let s1, leftTy = synth ctorEnv recEnv ctx env left
    let s2, rightTy = synth ctorEnv recEnv ctx (applyEnv s1 env) right
    let a = freshVar()
    let b = freshVar()
    let c = freshVar()
    let s3 = unifyWithContext ctx [] span (apply s2 leftTy) (TArrow(b, c))
    let s4 = unifyWithContext ctx [] span (apply s3 rightTy) (TArrow(a, apply s3 b))
    let finalS = compose s4 (compose s3 (compose s2 s1))
    (finalS, TArrow(apply s4 a, apply finalS c))
```

**Eval.fs (evaluation):** Add eval cases in the `eval` function, after the `SetField` case.

For `PipeRight` — evaluate as `f(x)`:

| PipeRight (left, right, _) ->
    let argVal = eval recEnv moduleEnv env left
    let funcVal = eval recEnv moduleEnv env right
    match funcVal with

For `ComposeRight` — create a closure `fun x -> g(f(x))`:
```fsharp
| ComposeRight (left, right, _) ->
    let fVal = eval recEnv moduleEnv env left
    let gVal = eval recEnv moduleEnv env right
    // Create a closure that applies f then g
    let param = "__compose_x"
    let body = App(right, App(left, Var(param, unknownSpan), unknownSpan), unknownSpan)
    FunctionValue(param, body, env)
```

For `ComposeLeft` — create a closure `fun x -> f(g(x))`:
```fsharp
| ComposeLeft (left, right, _) ->
    let fVal = eval recEnv moduleEnv env left
    let gVal = eval recEnv moduleEnv env right
    // Create a closure that applies g then f
    let param = "__compose_x"
    let body = App(left, App(right, Var(param, unknownSpan), unknownSpan), unknownSpan)
    FunctionValue(param, body, env)
```

IMPORTANT for ComposeRight/ComposeLeft: The closure approach uses AST nodes + env capture. The `fVal` and `gVal` bindings are intentional — they force evaluation of both sides. But the body references `left` and `right` as Expr nodes which get re-evaluated from `env`. This works because `env` captures the current scope. However, to avoid re-evaluation issues, a better approach is to store evaluated values directly:

```fsharp
| ComposeRight (left, right, _) ->
    let fVal = eval recEnv moduleEnv env left
    let gVal = eval recEnv moduleEnv env right
    let param = "__compose_x"
    let fName = "__compose_f"
    let gName = "__compose_g"
    let body = App(Var(gName, unknownSpan), App(Var(fName, unknownSpan), Var(param, unknownSpan), unknownSpan), unknownSpan)
    let closureEnv = env |> Map.add fName fVal |> Map.add gName gVal
    FunctionValue(param, body, closureEnv)
```

Use the same pattern for `ComposeLeft` but swap f and g in the body:
```fsharp
| ComposeLeft (left, right, _) ->
    let fVal = eval recEnv moduleEnv env left
    let gVal = eval recEnv moduleEnv env right
    let param = "__compose_x"
    let fName = "__compose_f"
    let gName = "__compose_g"
    let body = App(Var(fName, unknownSpan), App(Var(gName, unknownSpan), Var(param, unknownSpan), unknownSpan), unknownSpan)
    let closureEnv = env |> Map.add fName fVal |> Map.add gName gVal
    FunctionValue(param, body, closureEnv)
```
