Add return type annotation productions. The pattern is: after params, a `COLON TypeExpr` before `EQUALS` wraps the body in `Annot(body, typeExpr, span)`.

**A. Decl level — add these productions (inline + indented for each):**

1. No params, return type only: `let x : int = 42`
```
// Phase 64: let with return type annotation (no params)
| LET IDENT COLON TypeExpr EQUALS SeqExpr
    { LetDecl($2, Annot($6, $4, ruleSpan parseState 1 6), ruleSpan parseState 1 6) }
| LET IDENT COLON TypeExpr EQUALS INDENT SeqExpr DEDENT
    { LetDecl($2, Annot($7, $4, ruleSpan parseState 1 8), ruleSpan parseState 1 8) }
```

2. MixedParamList + return type: `let f (x : int) y : bool = ...`
```
// Phase 64: let with mixed params + return type
| LET IDENT MixedParamList COLON TypeExpr EQUALS SeqExpr
    { let body = Annot($7, $5, ruleSpan parseState 1 7)
      let lambda = desugarMixedParams $3 body (ruleSpan parseState 1 7)
      LetDecl($2, lambda, ruleSpan parseState 1 7) }
| LET IDENT MixedParamList COLON TypeExpr EQUALS INDENT SeqExpr DEDENT
    { let body = Annot($8, $5, ruleSpan parseState 1 9)
      let lambda = desugarMixedParams $3 body (ruleSpan parseState 1 9)
      LetDecl($2, lambda, ruleSpan parseState 1 9) }
```
