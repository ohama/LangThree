Read each file and rewrite dot-notation calls to module function calls. Replacements:

HashSet:
- `HashSet ()` -> `HashSet.create ()`
- `hs.Add(arg)` -> `HashSet.add hs arg`
- `hs.Contains(arg)` -> `HashSet.contains hs arg`
- `hs.Count` -> `HashSet.count hs`

Queue:
- `Queue ()` -> `Queue.create ()`
- `q.Enqueue(arg)` -> `Queue.enqueue q arg`
- `q.Dequeue()` or `q.Dequeue ()` -> `Queue.dequeue q ()`
- `q.Count` -> `Queue.count q`

IMPORTANT for `queue-error.flt`: The expected error message is `"caught: Queue.Dequeue: queue is empty"`. This error comes from the `queue_dequeue` builtin in Eval.fs (NOT from the dot dispatch code), so the expected output does NOT change.

IMPORTANT: `HashSet.add hs arg` returns `bool` (same as `hs.Add(arg)`). `Queue.enqueue q arg` returns `()` (same as `q.Enqueue(arg)`).

IMPORTANT: Do NOT change expected output sections.
