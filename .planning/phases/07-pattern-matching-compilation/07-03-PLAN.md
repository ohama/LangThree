**Test categories:**

1. **ADT Constructor Patterns (PMATCH-01)**
   - Simple constructor match (None/Some):
     ```
     type Option = None | Some of int
     let x = Some 42
     let result = match x with
       | None -> 0
       | Some n -> n
     ```
     Expect: 42

   - Multi-constructor match (3+ constructors):
     ```
     type Color = Red | Green | Blue
     let c = Green
     let result = match c with
       | Red -> 1
       | Green -> 2
       | Blue -> 3
     ```
     Expect: 2

   - Constructor with nested data:
     ```
     type Tree = Leaf | Node of Tree * int * Tree
     let t = Node (Leaf, 5, Node (Leaf, 3, Leaf))
     let result = match t with
       | Leaf -> 0
       | Node (_, v, _) -> v
     ```
     Expect: 5

2. **Nested Patterns (PMATCH-01, PMATCH-02)**
   - Deep nested constructor:
     ```
     type Tree = Leaf | Node of Tree * int * Tree
     let t = Node (Node (Leaf, 1, Leaf), 2, Leaf)
     let result = match t with
       | Node (Node (_, x, _), y, _) -> x + y
       | Node (Leaf, y, _) -> y
       | Leaf -> 0
     ```
     Expect: 3

   - Nested with variable binding at all levels:
     ```
     type Pair = MkPair of int * int
     type Wrapper = Wrap of Pair
     let w = Wrap (MkPair (10, 20))
     let result = match w with
       | Wrap (MkPair (a, b)) -> a + b
     ```
     Expect: 30

3. **List Patterns (PMATCH-02)**
