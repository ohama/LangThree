  <action>
Find the initialBuiltinEnv list in Eval.fs. Locate the section with array builtins (search for "array_map" or "array_fold"). Add these two new builtins near the other list/array builtins.

For list_sort_by, use the same callValue pattern as array_map (Eval.fs lines ~567-573). The key function is a user closure that must be invoked via callValue (or callValueRef — check which is used in array_map).

Add to initialBuiltinEnv:

```fsharp
// list_sort_by : ('a -> 'b) -> 'a list -> 'a list
"list_sort_by", BuiltinValue (fun fVal ->
    BuiltinValue (fun listVal ->
        match listVal with
        | ListValue xs ->
            let keyed = xs |> List.map (fun x -> (callValue fVal x, x))
            let sorted = keyed |> List.sortWith (fun (k1, _) (k2, _) -> Value.valueCompare k1 k2)
            ListValue (sorted |> List.map snd)
        | _ -> failwith "list_sort_by: expected list"))

// list_of_seq : seq<'a> -> 'a list
"list_of_seq", BuiltinValue (fun v ->
    match v with
    | ListValue xs        -> ListValue xs
    | ArrayValue arr      -> ListValue (Array.toList arr)
    | HashSetValue hs     -> ListValue (hs |> Seq.toList)
    | QueueValue q        -> ListValue (q |> Seq.toList)
    | MutableListValue ml -> ListValue (ml |> Seq.toList)
    | _ -> failwith "list_of_seq: expected a collection (list, array, HashSet, Queue, or MutableList)")
```

IMPORTANT: If array_map uses callValueRef instead of callValue (check the actual Eval.fs source), use the same pattern for list_sort_by. The STATE.md confirms "callValueRef forward reference pattern used for builtins that invoke user closures" — so use callValueRef.

In the same initialBuiltinEnv section, add two more builtins immediately after or near the list builtins from Task 1.

array_sort sorts the ArrayValue in-place using System.Array.Sort with Value.valueCompare. Returns TupleValue [] (unit), consistent with array_set behavior.

array_of_seq creates a new ArrayValue from any collection type, using the same pattern as list_of_seq.

Add to initialBuiltinEnv:

```fsharp
// array_sort : 'a array -> unit
"array_sort", BuiltinValue (fun arrVal ->
    match arrVal with
    | ArrayValue arr ->
        System.Array.Sort(arr, fun x y -> Value.valueCompare x y)
        TupleValue []
    | _ -> failwith "array_sort: expected array")

// array_of_seq : seq<'a> -> 'a array
"array_of_seq", BuiltinValue (fun v ->
    match v with
    | ListValue xs        -> ArrayValue (List.toArray xs)
    | ArrayValue arr      -> ArrayValue (Array.copy arr)
    | HashSetValue hs     -> ArrayValue (hs |> Seq.toArray)
    | QueueValue q        -> ArrayValue (q |> Seq.toArray)
    | MutableListValue ml -> ArrayValue (ml |> Seq.toArray)
    | _ -> failwith "array_of_seq: expected a collection")
```
