    ```fsharp
    | "EndsWith" ->
        BuiltinValue (fun arg ->
            match arg with
            | StringValue suffix -> BoolValue (s.EndsWith(suffix))
            | _ -> failwith "String.EndsWith: expected string argument")
    | "StartsWith" ->
        BuiltinValue (fun arg ->
            match arg with
            | StringValue prefix -> BoolValue (s.StartsWith(prefix))
            | _ -> failwith "String.StartsWith: expected string argument")
    | "Trim" ->
        BuiltinValue (fun arg ->
            match arg with
            | TupleValue [] -> StringValue (s.Trim())
            | _ -> failwith "String.Trim: takes no arguments (call as .Trim())")
    ```

    CRITICAL: Trim MUST return a BuiltinValue accepting TupleValue [], NOT a StringValue directly. `" hi ".Trim()` parses as App(FieldAccess(s,"Trim",_), Tuple([],_), _) — the FieldAccess result is the function position in App. Returning StringValue directly would cause "attempted to call non-function" at runtime.

    In Bidir.fs, extend the `| TString ->` match in the FieldAccess arm (around line 519). After the existing "Contains" case and before the `| _ -> raise` fallback, add:

    ```fsharp

    | "StartsWith" -> (s1, TArrow(TString, TBool))
    | "Trim" -> (s1, TArrow(TTuple [], TString))
    ```

    Do NOT add eprintfn to Eval.fs in this task — that is Task 2.

    In Eval.fs, after the existing `applyPrintfnArgs` function (around line 102), add:

    ```fsharp
    /// Build a curried BuiltinValue chain for eprintfn (like applyPrintfnArgs but writes to stderr).
    let rec applyEprintfnArgs (fmt: string) (remaining: string list) (collected: Value list) : Value =
        match remaining with
        | [] ->
            let result = substitutePrintfArgs fmt (List.rev collected)
            stderr.Write(result)
            stderr.Write("\n")
            stderr.Flush()
            TupleValue []
        | _ :: rest ->
            BuiltinValue (fun argVal ->
                applyEprintfnArgs fmt rest (argVal :: collected))
    ```

    In Eval.fs initialBuiltinEnv, add `eprintfn` alongside the existing `printfn` entry:

    "eprintfn", BuiltinValue (fun fmtVal ->

            applyEprintfnArgs fmt specifiers []
        | _ -> failwith "eprintfn: first argument must be a format string")
