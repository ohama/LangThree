  <action>
Create `src/LangThree/ProjectFile.fs` with module `ProjectFile`. Add it to LangThree.fsproj BEFORE `Cli.fs` (between Repl.fs and Cli.fs, as it has no dependency on Cli but Program.fs uses both).

The module must contain:

1. **TOML POCO types** (all with `[<CLIMutable>]` -- required for Tomlyn reflection-based deserialization):
   - `TomlProjectSection`: mutable fields `name: string`, `prelude: string`
   - `TomlTarget`: mutable fields `name: string`, `main: string`
   - `TomlFunProj`: mutable fields `project: TomlProjectSection`, `executable: System.Collections.Generic.List<TomlTarget>`, `test: System.Collections.Generic.List<TomlTarget>`
   
   IMPORTANT: Field names must match TOML keys exactly (lowercase). `[[executable]]` maps to field `executable` (singular, NOT `executables`).

2. **F# record types** for idiomatic consumption:
   - `TargetConfig`: `Name: string`, `Main: string` (absolute path)
   - `FunProjConfig`: `ProjectName: string`, `PreludePath: string option`, `Executables: TargetConfig list`, `Tests: TargetConfig list`

3. **Helper function** `makeTarget (projDir: string) (t: TomlTarget) : TargetConfig` -- resolves `t.main` relative to projDir using `Path.GetFullPath(Path.Combine(projDir, t.main))`. Null-check `t.name` and `t.main`.

4. **`parseFunProj (tomlText: string) (projDir: string) : FunProjConfig`** -- calls `TomlSerializer.Deserialize<TomlFunProj>(tomlText)`, converts to FunProjConfig. Null-check `raw.project` using `box projSection <> null`. PreludePath is `Some absolutePath` only when non-null and non-empty, else `None`.

5. **`findFunProj () : string option`** -- checks `Path.GetFullPath("funproj.toml")`, returns `Some path` if exists, else `None`.

6. **`loadFunProj (path: string) : Result<FunProjConfig, string>`** -- reads file, calls parseFunProj with `Path.GetDirectoryName path` as projDir. Wraps exceptions in `Error`.

Open `System.IO` and `Tomlyn` at the top.

Follow the exact code patterns from the research document (Pattern 1, 2, and the Complete ProjectFile.fs Module section).
  </action>
  <verify>`dotnet build src/LangThree/LangThree.fsproj -c Release` succeeds with no errors</verify>
  <done>ProjectFile.fs compiles, exports parseFunProj/findFunProj/loadFunProj/FunProjConfig/TargetConfig</done>
</task>
