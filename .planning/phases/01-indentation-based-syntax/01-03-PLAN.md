  <name>Add indent width validation and improve error message formatting</name>
  <files>
    src/LangThree/IndentFilter.fs
  </files>
  <action>
    1. Add validateIndentWidth function:
       ```fsharp
       let validateIndentWidth (config: IndentConfig) (col: int) (lineNum: int) : unit =
           if config.StrictWidth && col > 0 && col % config.IndentWidth <> 0 then
               let expected = (col / config.IndentWidth + 1) * config.IndentWidth
               raise (IndentationError(lineNum,
                   sprintf "Indentation must be multiple of %d spaces. Found %d spaces, expected %d"
                           config.IndentWidth col expected))
       ```
       Call this in processNewline BEFORE any indent stack operations.

    2. Add formatExpectedIndents helper function:
       ```fsharp
       let formatExpectedIndents (config: IndentConfig) (stack: int list) : string =
           let validLevels = stack |> List.map string |> String.concat ", "
           let nextIndent = (List.head stack) + config.IndentWidth
           sprintf "Expected one of: [%s] (to match previous level) or %d (to start new block)"
                   validLevels nextIndent
       ```

    3. Update IndentationError messages in processNewline:
       - When col doesn't match any level in stack:
         ```fsharp
         raise (IndentationError(state.LineNum,
             sprintf "Invalid indentation at line %d: found %d spaces.\n%s"
                     state.LineNum col (formatExpectedIndents config state.IndentStack)))
         ```

    4. Update processNewlineWithContext to use same error format:
       - For match pipe misalignment:
         ```fsharp
         raise (IndentationError(state.LineNum,
             sprintf "Match pattern pipe must align with 'match' keyword at column %d, found at column %d"
                     matchCol col))
         ```

    Implementation note: Use sprintf instead of string interpolation ($"") for consistency with existing F# codebase conventions. Include line number prominently at start of error message.
