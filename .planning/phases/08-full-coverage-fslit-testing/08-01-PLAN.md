- `2 * 4` -> Multiply (Number 2, Number 4)
- `10 / 2` -> Divide (Number 10, Number 2)

**Comparison:**
- `1 = 2` -> Equal (Number 1, Number 2)
- `1 <> 2` -> NotEqual (Number 1, Number 2)
- `1 < 2` -> LessThan (Number 1, Number 2)
- `1 > 2` -> GreaterThan (Number 1, Number 2)
- `1 <= 2` -> LessEqual (Number 1, Number 2)
- `1 >= 2` -> GreaterEqual (Number 1, Number 2)

**Boolean:**
- `true && false` -> And (Bool true, Bool false)
- `true || false` -> Or (Bool true, Bool false)

**If:**
- `if true then 1 else 2` -> If (Bool true, Number 1, Number 2)

**Let / LetRec:**
- `let x = 5 in x + 1` -> Let ("x", Number 5, Add (Var "x", Number 1))
- `let rec f x = f x in f 1` -> LetRec ("f", "x", App (Var "f", Var "x"), App (Var "f", Number 1))

**App:**
- `(fun x -> x) 42` -> App (Lambda ("x", Var "x"), Number 42)

**Match:**

**Tuple:**
- `(1, true, "hi")` -> Tuple [Number 1, Bool true, String "hi"]

**List / Cons:**
- `1 :: 2 :: []` -> Cons (Number 1, Cons (Number 2, EmptyList))

**Annotation:**
- `(1 + 2 : int)` -> Annot (Add (Number 1, Number 2), TEInt)

**LetPat (pattern destructuring):**
